# odin-tic-tac-toe
## Objective
A tic-tac-toe game for The Odin Project course
The purpose of this project is to practice using closures and factory functions.
This project aims to use closures and private variables for as much of the project as possible.

## Gameplay
A basic gameplay loop was completed, allowing two players.
The game requires 2 players to take turns placing their own marker.
The goal is to place 3 markers in a complete row before your opponent.

## Features
Each player can assign their own name.
Each player can change their assigned name at any point in the game.

## Possible updates
 - A color picker can be added for each player if they do not wish to keep the pre-determined colors.
 - The markers can be swapped so it does not necessarily need to be just naughts or crosses.
 - The gameplay loop can be limited to best out 3, or best out of 5 before requiring to reset the scores.
